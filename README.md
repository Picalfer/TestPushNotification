# Push Notification Test

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)
[![Firebase](https://img.shields.io/badge/Service-Firebase_Messaging-orange.svg)](https://firebase.google.com/docs/cloud-messaging)

Тестовое приложение для изучения Firebase Cloud Messaging (FCM) в Android. Демонстрирует работу с push-уведомлениями и обработку полученных сообщений.

## Функциональность

- Получение и отображение FCM токена
- Обработка входящих push-сообщений
- BroadcastReceiver для коммуникации между сервисом и активностью
- Показ Toast при получении сообщения

## Технологии

- Kotlin
- Firebase Cloud Messaging
- BroadcastReceiver
- FirebaseMessagingService
